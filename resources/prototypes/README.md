# Prototypes

Minimal, self-contained examples demonstrating the key technologies used in this project.

| Prototype | Description |
|---|---|
| [neo4j/](neo4j/) | Connect to Neo4j, create Person nodes + PARENT_OF relationship, query children — Python, JavaScript, Ruby |
| [gedcom/](gedcom/) | Parse a GEDCOM 5.5.1 string with JavaScript and print individuals + birth dates |
| [visualization/](visualization/) | Interactive force-directed family tree with D3.js v7 — open directly in a browser |
